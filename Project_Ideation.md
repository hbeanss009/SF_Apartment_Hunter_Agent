Q1. What do I want this project to accomplish?
This project will take inputs from the user mainly city, price, user preferences and recommend listings to the user. 

Q2. How would this be done? 
This would done in 3 steps - discover, analyse and recommend listings to the user. 
 
Q3. Do I want to use agents for all three steps? 

1. Discover (first step) - workflow 

The first step of the process discovery is a deterministic flow and since there is a defined output and sequence of steps this goal can be achieved through a workflow. 

2. Analyse (second step) - agent 
Analysing the listings involves a bit of non-deterministic and deterministic tasks and will need to be an agentic.

3. Recommend (third step) - agentic workflow 
The process of ranking the listings based on certain criteria is deterministic and can be a workflow, however I would also want this agent to be able to answer any questions user may have on why certain listings were ranked higher and if user makes additional requests of wanting additional features, agent should be able to accomodate that as well.
