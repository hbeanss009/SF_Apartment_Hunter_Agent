# ApartmentHuntingAgent
Agent that scrapes, analyses and recommends apartment listings based on user criteria

This project will take inputs from the user mainly minimum bed/bath, budget range, must have amenities, nice-to-have amenities and recommend a list of apartments to the user. 

This would done in 3 steps - discover, analyse and recommend listings to the user.

1. Discover (workflow) - The first step of the process discovery is a deterministic flow and since there is a defined output and sequence of steps this goal can be achieved through a workflow.

2. Analyse (agent) - Analysing the listings involves a bit of non-deterministic and deterministic tasks and will need to be an agentic.

3. Recommend (agentic workflow) - The process of ranking the listings based on certain criteria is deterministic and can be a workflow, however I would also want    this agent to be able to answer any questions user may have on why certain listings were ranked higher and if user makes   additional requests of wanting additional features, agent should be able to accomodate that as well.
